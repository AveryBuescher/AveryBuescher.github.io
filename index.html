Pro-Calendar

This project was completed as part of a team of 5 for the wake tech capstone project. During the course of the semester, we worked together to design, build, test, and finally present an application based on a provided software requirements specification document. The Pro calendar is a calendar application that aids users with time management. It allows users to create and view events, integrate and synchronize with their google calendar, and categorize their events in order to view metrics on how they spend their time. This project was created using python and the Django framework, as well as HTML, CSS, and a small amount of JavaScript. While it may be your average "To-Do" app on the surface, this project provided in-depth experience with the software development life cycle, version control, and working in a team for an extended project.
